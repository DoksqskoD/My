-Экспортируем с Qt designer из формата .ui в .py

-Устанавливаем библиотеку pyqt5 и pyqt5-tools
_____________________________________________________
-Прописываем в терминале
pip install pyqt5
pip install pyqt5-tools
-Если не получаеетца установить pyqt5-tools устанавиливай pyqt-tools и так будет работать
_____________________________________________________

-Проверям установились все нужные библиотеки
_____________________________________________________
-Прописываем в терминале
pyuic5
-Терминал должен выдвть ошибку...
Error: one input ui-file must be specified
_____________________________________________________
-Если все хоошо тогда прописываем

-pyuic5 -x (Имя файла который ты создал с помощью Qt designer).ui -o (Имя файла который ты создаешь).py
